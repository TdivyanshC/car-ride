# Car Ride App Design System

## Overview
Modernizing the UI to match professional ride-sharing apps like Uber and Ola, with seamless interactions and contemporary design.

## Color Palette
- Primary: #276EF1 (Uber-style blue)
- Secondary: #1A73E8
- Background: #FFFFFF (light) or #121212 (dark option)
- Surface: #F8F9FA
- Text Primary: #1A1A1A
- Text Secondary: #666666
- Accent: #34C759 (success green)
- Error: #FF3B30
- Warning: #FF9500

## Typography
- Font Family: System font stack (Inter or SF Pro for iOS)
- Headings: 24px bold, 20px semibold
- Body: 16px regular
- Caption: 14px regular, 12px for small text

## Components
### Buttons
- Primary: Rounded, full width, primary color background
- Secondary: Outlined, primary color border
- Floating: Circular, elevated

### Cards
- Rounded corners (12px)
- Elevation with shadows
- Blur effects for overlays

### Inputs
- Rounded text inputs with icons
- Floating labels
- Autocomplete dropdowns

### Map Integration
- Full-screen map background
- Floating cards for inputs
- Markers with custom icons

## Layout Patterns
### Search Screen
- Full-screen MapView
- Bottom card with location inputs
- Floating search button

### Home Screen
- Header with greeting
- Stats cards in grid
- Action cards with icons
- Recent activity list

### Forms
- Clean, spaced inputs
- Validation feedback
- Progressive disclosure

## Animations
- Smooth transitions (300ms)
- Scale on press
- Slide in/out for modals
- Fade for loading states

## Responsiveness
- Mobile-first design
- Safe area handling
- Keyboard avoidance

## Accessibility
- High contrast ratios
- Touch targets 44px minimum
- Screen reader support
